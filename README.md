# 🎉 Inquiry System – 問い合わせ管理アプリ
 このアプリは、  
- ユーザーが「氏名／社員番号／ジャンル／内容」を入力して問い合わせを送信  
- 管理者が問い合わせの履歴を一覧で確認  
というフローで動作する、**シンプルな問い合わせ管理システム**です。

🚀 Features（機能一覧）
✏️ 1. 問い合わせ登録

名前・従業員番号・種別・内容を入力

入力確認 → 完了画面の 2ステップ構造

必須項目バリデーション対応

📜 2. 問い合わせ履歴

自分の問い合わせを一覧で確認

ジャンルでフィルタリング可能

新しい順に表示

🔐 3. 管理者向け機能

すべての問い合わせを一覧表示

ユーザーごとに問い合わせを参照
